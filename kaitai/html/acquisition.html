Type: AcqSvProfile

Profile for a specific SV for debugging purposes. The message describes SV profile during acquisition time. The message is used to debug and measure the performance.

OffsetSizeIDTypeNote
0 ... job_type u1 SV search job type (deep, fallback, etc)
1 ... status u1 Acquisition status 1 is Success, 0 is Failure
2 ... cn0 u2le CN0 value. Only valid if status is '1'
4 ... int_time u1 Acquisition integration time
5 ... sid GnssSignal GNSS signal for which acquisition was attempted
7 ... bin_width u2le Acq frequency bin width
9 ... timestamp u4le Timestamp of the job complete event
13 ... time_spent u4le Time spent to search for sid.code
17 ... cf_min s4le Doppler range lowest frequency
21 ... cf_max s4le Doppler range highest frequency
25 ... cf s4le Doppler value of detected peak. Only valid if status is '1'
29 ... cp u4le Codephase of detected peak. Only valid if status is '1'

Type: MsgAcqSvProfile

Type: MsgAcqResult

This message describes the results from an attempted GPS signal acquisition search for a satellite PRN over a code phase/carrier frequency range. It contains the parameters of the point in the acquisition search space with the best carrier-to-noise (CN/0) ratio.

OffsetSizeIDTypeNote
0 ... cn0 f4le CN/0 of best point
4 ... cp f4le Code phase of best point
8 ... cf f4le Carrier frequency of best point
12 ... sid GnssSignal GNSS signal for which acquisition was attempted
