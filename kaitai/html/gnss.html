Type: CarrierPhase

Carrier phase measurement in cycles represented as a 40-bit fixed point number with Q32.8 layout, i.e. 32-bits of whole cycles and 8-bits of fractional cycles. This phase has the same sign as the pseudorange.

OffsetSizeIDTypeNote
0 ... i s4le Carrier phase whole cycles
4 ... f u1 Carrier phase fractional part

Type: GpsTimeDep

A wire-appropriate GPS time, defined as the number of milliseconds since beginning of the week on the Saturday/Sunday transition.

OffsetSizeIDTypeNote
0 ... tow u4le Milliseconds since start of GPS week
4 ... wn u2le GPS week number

Type: GnssSignal

Signal identifier containing constellation, band, and satellite identifier.

OffsetSizeIDTypeNote
0 ... sat u1 Constellation-specific satellite id. For GLO can either be (100+FCN) where FCN is in [-7,+6] or the Slot ID in [1,28].
1 ... code u1 Signal constellation, band and code

Type: GpsTime

A wire-appropriate receiver clock time, defined as the time since the beginning of the week on the Saturday/Sunday transition. In most cases, observations are epoch aligned so ns field will be 0.

OffsetSizeIDTypeNote
0 ... tow u4le Milliseconds since start of GPS week
4 ... ns_residual s4le Nanosecond residual of millisecond-rounded TOW (ranges from -500000 to 500000)
8 ... wn u2le GPS week number

Type: GpsTimeSec

A GPS time, defined as the number of seconds since beginning of the week on the Saturday/Sunday transition.

OffsetSizeIDTypeNote
0 ... tow u4le Seconds since start of GPS week
4 ... wn u2le GPS week number

Type: SvId

A (Constellation ID, satellite ID) tuple that uniquely identifies a space vehicle.

OffsetSizeIDTypeNote
0 ... sat_id u1 Constellation-specific satellite id. For GLO can either be (100+FCN) where FCN is in [-7,+6] or the Slot ID in [1,28].
1 ... constellation u1 Constellation ID to which the SV belongs
