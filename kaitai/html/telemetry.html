Telemetry format specification

Type: Telemetry

OffsetSizeIDTypeNote

Type: TelemetrySv

OffsetSizeIDTypeNote
0 ... az u1 Azimuth angle (range 0..179)
1 ... el s1 Elevation angle (range -90..90)
2 ... availability_flags u1 Observation availability at filter update
3 ... pseudorange_residual s2le Pseudorange observation residual
5 ... phase_residual s2le Carrier-phase or carrier-phase-derived observation residual
7 ... outlier_flags u1 Reports if observation is marked as an outlier and is excluded from the update
8 ... ephemeris_flags u1 Ephemeris metadata
9 ... correction_flags u1 Reserved
10 ... sid GnssSignal GNSS signal identifier (16 bit)

Type: MsgTelSv

This message includes telemetry pertinent to satellite signals available to Starling.

OffsetSizeIDTypeNote
0 ... wn u2le GPS week number
2 ... tow u4le GPS Time of Week
6 ... n_obs u1 Total number of observations. First nibble is the size of the sequence (n), second nibble is the zero-indexed counter (ith packet of n)
7 ... origin_flags u1 Flags to identify the filter type from which the telemetry is reported from
8 ... sv_tel TelemetrySv Array of per-signal telemetry entries
