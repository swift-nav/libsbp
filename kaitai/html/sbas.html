Sbas format specification

Type: Sbas

OffsetSizeIDTypeNote

Type: MsgSbasRaw

This message is sent once per second per SBAS satellite. ME checks the parity of the data block and sends only blocks that pass the check.

OffsetSizeIDTypeNote
0 ... sid GnssSignal GNSS signal identifier.
2 ... tow u4le GPS time-of-week at the start of the data block.
6 ... message_type u1 SBAS message type (0-63)
7 ... data u1 Raw SBAS data field of 212 bits (last byte padded with zeros).
